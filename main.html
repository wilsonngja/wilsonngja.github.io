>My Projects_

Data Visualisation

Data Visualisation is a project done during my 3-months internship with Centre for Strategic Infocomm Technologies (CSIT). This project required HTML and Python prerequisite. Web scraping and data cleaning were conducted before data is being visualisation on a correlation graph with node representations.

Deep Learning for Human Motion Detection

This project is done in group as my final year project in SP which involves detecting human motion on Personal Mobility Devices (PMD). The inspiration was from the increasing trend of PMDs accidents in Singapore. YOLOv3 Framework is used for detection with images trained from scratch. My team also used Shi-Tomashi Corner Detection algorithm to identify edges to calculate speed of pedesterian on PMDs.

Smart Washing Machine

Smart Washing Machine System is an integrated project done by the whole class. The class are splitted into groups and assigned different washing machine components to program in C Language. My team were tasked to program a Stepper motor to simulate washing motion. At the end, the whole class integrated the task and demostrated to our lecturer.

Keyword Identifier

This is a side project that I programmed as a hobby in Python language. By making use to Natural Language Toolkit (NLTK) library to identify the keywords. Filtering is then carried out to remove stopwords and other unwanted data such as '\t' and '\n'. After the cleaning is done, keywords is highlighted by using 'Document' Library available.

Substitution Cipher

Substitution Cipher is a project assigned during my Data Structure and Algorithm (DSA) class. The deliverables are to conduct encryption and decryption from a source file. Encryption is done by changing each alphabet of a text and adding a random assigned alphabets for every few alphabets. My team and I also added features by allowing user to encrypt from user input.

Smart Home GUI

Smart Home GUI was done as a project in my Object Oriented Programming (OOP) module. During this module, we programmed a simulation of a Graphical User Interface (GUI). Within my GUI, I made use of images, button and slider to create an interface which allows user to toggle home appliances. The coding was done in Java Language.
